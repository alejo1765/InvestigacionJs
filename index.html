¿Cómo son todas las formas de declaración de funciones en JS?

Una de la caracteristicas mas importantes de las funciones de JavaScript es su versatilidad. En JavaScript las funciones son diferentes a las funciones de otros lenguajes de programacion. En JavaScript las funciones son un objeto. Por ello se les puede asignar variable, arrays u otros objetos. La forma en la cual hallamos definido nuestra funcion, definira el comportmiento de la misma. Siendo la funcion que podemos crear en JavaScript son los siguientes

  • Function declaration
  • Function expression
  • IIFE (Inmediately Invoked function Expression)
  • Shorthand method definition
  • Arrow function
  • Generator function
  • Function constructor

¿Cuáles son los operadores y conversión entre tipos?

Existen varios tipos de operadores:

Operadores arimetricos:

  • Suma: 4+4=8
  • Resta:4+2=2
  • Multiplicación: 3*3=9
  • División 20/2=10
  • Modulo: 5%2=1

Operadores relacionales

  • > Mayor que
  • < Menor que
  • >= Mayor o igual
  • <= Menor o igual
  • == Igual que sin tener en cuenta el tipo de dato
  • === igual que teniendo en cuenta el tipo de dato
  • != Diferente que sin tener en cuenta el tipo de dato
  • !== Doble diferente teniendo en cuenta el tipo de dato

Operadoresde incremento y decremento

  • Incremento: i+=x la variable i incrementa x
  • Decremento: i-= x la variable i decrementa x
  • Division: i/= x la variable i se divide por x
  • Multiplicacion: i*= x la variable i se multiplica por x
  • Operador unario incremento: i++ la variable i incrementa 1
  • Operador unario decremento: i- - la variable i decrementa 1

Operadores logicos:

  • !: negación not/no

¿Cuáles son las estructura de control de JS?

Las estructuras de control son indispensables en cualquier tipo de lenguaje de programacion que se jacte de serlo. Las estructuras de control sirven para controlar el flujo de ejecucion de un algoritmo, que pase tal situacion segun tal condicion.

Existen las siguientes extruncturas de control en js:

  • If/else
  • Switch
  • Bucle for
  • Bucle while
  • Bucle do/while

¿Qué es el objeto dom en JS? Y ¿Cómo trabaja?

Un objeto map cumple una función muy similar a la de los objetos normales, la cual es almacenar pares clave-valor. Aunque cuenta con algunas diferencias. En los objetos normales las claves solo pueden ser de tipo string o symbol; en el caso de map pueden ser de cualquier tipo, con map podemos conocer su tamaño utilizando la propiedad size, y un map es iterable sin necesidad de las claves como si se necesitan con los objetos normales.
