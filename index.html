¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?

las variables se pueden declarar usando "var", "const" o "let". "var" tiene alcance de funcion mientras que "const" y "let" tiene alcance de un bloque. Las variables "const" no pueden ser reasignadas, mientras que las variables ler si pueden serlo, "var", "const" y "let" pueden ser confusas al principio. Todos los lenguajes de programacion tienen estucturas de datos integradas. Estas se pueden utilizar para costruir otras estructuras de datos. Siempre que sea posible, se hacen comparacione de otros lenguajes. Las expresiones regulares son patrones que se utilizan para hacer coincidir combinaciones de caracteres en cadenas. En JavaScript, las expresiones regulares tambien son objetos

¿Cómo son todas las formas de declaración de funciones en JS?

Una de la caracteristicas mas importantes de las funciones de JavaScript es su versatilidad. En JavaScript las funciones son diferentes a las funciones de otros lenguajes de programacion. En JavaScript las funciones son un objeto. Por ello se les puede asignar variable, arrays u otros objetos. La forma en la cual hallamos definido nuestra funcion, definira el comportmiento de la misma. Siendo la funcion que podemos crear en JavaScript son los siguientes

  • Function declaration
  • Function expression
  • IIFE (Inmediately Invoked function Expression)
  • Shorthand method definition
  • Arrow function
  • Generator function
  • Function constructor

¿Cuáles son los operadores y conversión entre tipos?

Existen varios tipos de operadores:

Operadores arimetricos:

  • Suma: 4+4=8
  • Resta:4+2=2
  • Multiplicación: 3*3=9
  • División 20/2=10
  • Modulo: 5%2=1

Operadores relacionales

  • > Mayor que
  • < Menor que
  • >= Mayor o igual
  • <= Menor o igual
  • == Igual que sin tener en cuenta el tipo de dato
  • === igual que teniendo en cuenta el tipo de dato
  • != Diferente que sin tener en cuenta el tipo de dato
  • !== Doble diferente teniendo en cuenta el tipo de dato
